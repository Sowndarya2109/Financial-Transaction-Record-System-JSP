OVERVIEW:
This is a Java web application developed using Servlets, JSP, JDBC, and MySQL.
It allows users to add and view financial transaction records.

FEATURES:
Add new transaction

View a specific transaction

View all transactions

Input validation (amount, account holder, transaction type)

Duplicate record checking

HOW TO RUN:
Import project into Eclipse.

Configure database connection.

Deploy on Tomcat server.

Run in browser.
